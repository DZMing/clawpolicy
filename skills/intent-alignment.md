# Intent Alignment Skill

This skill helps OpenClaw adapt to your working style by learning from task outcomes, workflow choices, and feedback loops.

## Capabilities

- Preference learning from recent tasks and repository history
- Pattern detection for tooling, workflow, and risk tolerance
- Incremental adaptation with explicit user override

## Safety Boundaries

- Never override explicit user constraints
- Never run destructive actions without confirmation
- Keep all local preference data private
